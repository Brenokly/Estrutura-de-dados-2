Esse pacote está destinado para guardar todas as classes que componhe a estrutura da tabela hash usada no trabalho.

Onde:

 - CollisionTreatment.java é apenas uma ENUM (enumeração) criada para definir os tipos de tratamento de colisão que uma tabela pode ter.

 - HashFunctions.java como o próprio nome já diz, é a classe, abstrata, que define as funções hashs que uma tabela pode usar para realizar ah
  dispersão das chaves.

 - HashTable é a própria criação da classe para tabela hash. Ela é configurável, não possível todas as configurações possíveis que uma tabela hash
  poder ter, mas dá para se divertir com ela.

 - HashType é outro ENUM (enumeração), mas ela defini os tipos de funções hashs existentes.
